## 2. 1-D Data handling Using Pandas - Series
### Series?

-  1-D labeled **array**
-  possible to indicate index


```python
s = pd.Series([1,4,9,16,25])
s
```




    0     1
    1     4
    2     9
    3    16
    4    25
    dtype: int64




```python
t = pd.Series({'one':1,'two':2,'three':3,'four':4,'five':5})
t
```




    one      1
    two      2
    three    3
    four     4
    five     5
    dtype: int64



### Series + Numpy
-  Series is similar to ndarray


```python
s[1]
```




    4




```python
t[1]
```




    2




```python
t[1:3]
```




    two      2
    three    3
    dtype: int64




```python
s[ s > s.median()] # Extract only values bigger than own median values
```




    3    16
    4    25
    dtype: int64




```python
s[[3,1,4]]
```




    3    16
    1     4
    4    25
    dtype: int64




```python
import numpy as np
np.exp(s)
```





    dtype('int64')
